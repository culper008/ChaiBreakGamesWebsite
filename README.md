# Chai Break Games Website

Official landing page for **Chai Break Games** — India's next mobile game studio.

## Deployed to

- **chaibreak.games** (custom domain via GitHub Pages)

## Local Development

```bash
cd ChaiBreakGamesSite
# Serve locally (any HTTP server works)
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Deployment

Push to `main` branch. GitHub Pages automatically deploys to chaibreak.games (DNS configured).

## Content

- Studio intro & mission
- Jugaad Racer game showcase
- Gameplay mechanics
- Personas & target audience
- Visual direction (coming soon)
- Google Play pre-registration CTA
